Static shell package extracted from the WordPress theme.

Contents:
- `index.html`: shell-only page with header, side-nav, search overlay, and footer
- `css/`, `js/`, `img/`, `icon-library/`: required front-end assets

Usage:
1. Open `index.html` directly, or serve the `static-shell` folder from any static web server.
2. Edit navigation labels, department name, footer links, and logos directly in `index.html`.
3. Keep all asset folders next to `index.html`; the file paths are relative to this package.
